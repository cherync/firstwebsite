There are five webpages in this folder.
1. questions.js contains the questions for the quizzes on the website: the question and answer choices
2. q.html contains the commands for the the questions and answers, for example getting an "O" for the right answer and "X" for wrong
3. q1.html contains the questions for the Korean-American History
4. q2.html contains questions for Korean-American culture
5. q3.html contains questions for Korean language
